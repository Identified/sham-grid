@sham-grid

Oh noes! Not another grid!?!?.

Well, yes... and no. Is not a full-featured grid. It's actually a simpler way to get responsive and flexible versions of three most common layouts — a fat column with a sidebar, two equal columns, and three equal columns. You can use these patterns repeatedly to acheive anything you like.

Instead of complex naming conventions, you get three basic mixins: @sham-golden-wide/narrow, @sham-half, @sham-third.

Best of all everything scales beautifully for phones and tablets.

(Resize the browser window to see sham-grid in action.)

Installation

Naturally, you'll need Sass to use these mixins.

To install using Bower:

bower install sham-modal

Or include the main .scss file in your project

@import 'sham-modal';

We also set "box-sizing: border-box" so our padding has consistent results across browsers.

%sham-narrow

Extend %sham-narrow in the selector your choice to get a sidebar. You don't need special classes or secret code. In this case our sidebar is:

<aside>Sidebar stuff</aside>

The SCSS: 

aside {@extend %sham-narrow;}

%sham-wide

Extend %sham-wide on an element adjacent element to get the other part of you your golden ratio.

<section>Main stuff</section>

The SCSS: 

section#main {@extend %sham-wide;}

%sham-third

Extend %sham-third on three adjacent elements to get a three-column layout.

<div class="foo">Bar</div>

The SCSS: 

div.foo {@extend %sham-third;}

These columns are nested inside the wide column.

Since padding is used for gutters, you'll need interior elements if you want to add color to your columns, like we've done here:

<div class="foo"><div class="inner">Bar</div></div>

%sham-half

Extend %sham-half on two adjacent elements to get a two-column layout.

<div class="bar">Foo</div>

The SCSS: 

div.bar {@extend %sham-third;}

It's easy!

Bonus: a simple box! %sham-single-panel

In case you want a simple mobile-friendly box in the middle of your page.

<div class="baddabing">Baddaboom</div>

The SCSS: 

div.baddabing {@extend %sham-single-panel;}

About Sham

We here at Identified use automative-themed names for our repos. "sham" – derived from chamois, a cloth comonly used to polish cars — is our all-purpose toolbox containing SASS mixins and variables to help us quickly build web apps.
